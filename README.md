# Singlish to Sinhala Translator – Playwright Test Automation

## Project Overview
This project contains automated test cases written using **Playwright** to evaluate the accuracy and robustness of the **SwiftTranslator (Singlish → Sinhala)** web application.

The tests cover:
- Correct translation scenarios (PASS cases)
- Incorrect or weak translation scenarios (FAIL cases)
- Real-world Singlish usage
- Mixed language, numbers, symbols, and UI behavior

Tested system: https://www.swifttranslator.com/

---
